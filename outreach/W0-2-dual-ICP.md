**목적**: RR2 (드롭시퍼 segment LTV 자체 단축) 해소. 단일 segment 의존 → 이중 segment 분산.

## ICP 1 (1차 타겟) — Solo dropshipper / Shopify owner

| 속성 | 값 |
|---|---|
| 페르소나 | "Alex" — 25-35세, 첫 Shopify 스토어 운영 6-18개월, MRR $1k-10k |
| 위치 | US, UK, Canada, Australia (영어권) |
| 1주일 주요 작업 | 상품 리서치 (3-5h), 광고 운영, 주문 처리 |
| 사용 중인 도구 | Shopify, Spocket/CJDropshipping, Adspy, Niche Scraper |
| **Pain** | 트렌딩 상품 리서치 매일 1-2h, 늦게 발견하면 이미 경쟁 포화 |
| **우리 가치** | "월요일 아침 1,400+ 스토어에서 폭발 중인 상품 Top 5 리스트, 자동 발송" |
| 가격 민감도 | 매우 높음 ($19에 저항 0, $49에 저항 약간) |
| LTV 기대 | $57-114 (3-6개월 retention) |
| 채널 비중 | TikTok 60% / Instagram Reels 20% / Reddit 20% |

## ICP 2 (2차 타겟, LTV anchor) — E-commerce ops manager / store operator

| 속성 | 값 |
|---|---|
| 페르소나 | "Sam" — 30-45세, 자체 브랜드 Shopify Plus 또는 소규모 brand 운영, 직원 2-10명 |
| 위치 | US, UK, Canada, Australia, 일부 EU |
| 1주일 주요 작업 | 시장 조사, 경쟁사 모니터링, 신상품 기획, 가격 전략 |
| 사용 중인 도구 | Shopify Plus, Klaviyo, Triple Whale, BuiltWith |
| **Pain** | 카테고리 트렌드 파악 vs 데이터 분석 시간 |
| **우리 가치** | "1,400+ 동종 스토어에서 신상품·가격 분포 매주 자동 모니터링" |
| 가격 민감도 | 중간 ($49 적정, $99까지 저항 가능) |
| LTV 기대 | **$294-588 (12-24개월 retention)** |
| 채널 비중 | YouTube Shorts 30% / Reddit r/ecommerce 40% / X / LinkedIn 30% |

## 측정 KPI (segment별 분리)

| KPI | ICP 1 (드롭시퍼) 타겟 | ICP 2 (브랜드 운영자) 타겟 |
|---|---|---|
| Trial 가입 비율 | 70% | 30% |
| Paid 전환 | Starter $19 80% | Pro $49 70% |
| D+30 retention | ≥ 50% | ≥ 75% |
| D+90 retention | ≥ 25% | ≥ 60% |
| 환불률 | ≤ 30% | ≤ 15% |
